What is HYPERTEXT ?

Hyper text is the Hyper links that means TEXT WITHIN TEXT

What is MARKUP LANGUAGE ?

Markup language is the defined by the MARKED TAGS that means it is used to apply layout and formatting conventions to a text document by using tags.

Finally,we concluded that HTML is used to create web pages or web documents by using HTML tags.

There are 2 types of tags in HTML i.e; Container tags and Non-Container tags

Container tags are tags which have both starting tag and ending tag where as Non-Container tags contains only starting tag
